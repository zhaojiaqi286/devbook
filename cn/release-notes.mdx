#### 三大核心服务

**Web Search API - 基础检索服务**
- **功能**：直接获取结构化搜索结果
- **数据格式**：标题/摘要/链接等结构化信息
- **引擎支持**：多搜索引擎灵活切换
- **适用场景**：需要原始搜索数据的应用

**Chat Search - 问答增强服务**
- **功能**：将搜索结果融入大模型生成智能回答
- **特色**：标注网页结果来源，确保信息可追溯
- **优势**：结合搜索准确性和AI理解能力
- **适用场景**：智能问答、知识查询

**Search Agent - 搜索智能体**
- **功能**：根据用户query分析搜索意图
- **智能化**：自动进行多个query拆解
- **深度整合**：将搜索结果深度融入大模型
- **输出质量**：提供全面、有深度的回答
- **适用场景**：复杂查询、深度研究
