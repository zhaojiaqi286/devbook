## 账户注册

### 如何注册Z.AI平台账户？

**注册方式：**
1. **手机号注册**
   - 访问 https://bigmodel.cn
   - 点击"注册"按钮
   - 输入手机号码
   - 获取并输入验证码
   - 设置登录密码
   - 同意用户协议和隐私政策
   - 完成注册

2. **微信扫码注册**
   - 访问官网首页
   - 点击"微信扫码登录"
   - 使用微信扫描二维码
   - 在微信中确认授权
   - 补充必要信息完成注册

3. **邮箱注册**
   - 选择邮箱注册选项
   - 输入有效邮箱地址
   - 查收验证邮件
   - 点击邮件中的验证链接
   - 设置密码完成注册

### 注册时提示手机号已存在怎么办？

**可能原因：**
- 该手机号已经注册过账户
- 之前使用该手机号注册但忘记了

**解决方案：**
1. 尝试直接登录
2. 如果忘记密码，使用"忘记密码"功能重置
3. 联系客服确认账户状态
4. 如确实需要新账户，可使用其他手机号注册

### 收不到短信验证码怎么办？

**常见原因及解决方案：**

1. **网络延迟**
   - 等待1-2分钟再检查
   - 确保手机信号良好

2. **短信被拦截**
   - 检查手机垃圾短信箱
   - 将106开头的号码加入白名单
   - 关闭短信拦截功能

3. **手机号输入错误**
   - 确认手机号码输入正确
   - 注意区号和格式

4. **运营商问题**
   - 联系手机运营商确认
   - 尝试使用其他手机号

5. **频繁获取**
   - 避免频繁点击获取验证码
   - 等待60秒后重新获取

**替代方案：**
- 使用邮箱注册
- 使用微信扫码注册
- 联系客服人工协助

### 注册时提示邮箱格式错误？

**检查要点：**
- 确保邮箱地址包含@符号
- 检查域名部分是否正确
- 避免使用特殊字符
- 确保邮箱地址完整

**支持的邮箱类型：**
- 常见邮箱：QQ、163、126、Gmail、Outlook等
- 企业邮箱：公司域名邮箱
- 教育邮箱：edu.cn结尾的邮箱

## 账户登录

### 支持哪些登录方式？

**登录方式：**
1. **手机号登录**
   - 输入注册手机号
   - 输入密码或验证码
   - 点击登录

2. **邮箱登录**
   - 输入注册邮箱
   - 输入密码
   - 点击登录

3. **微信扫码登录**
   - 点击微信登录
   - 扫描二维码
   - 微信确认授权

4. **第三方登录**
   - 支持GitHub登录
   - 支持Google登录
   - 更多方式持续增加

### 忘记密码怎么办？

**密码重置步骤：**
1. 点击登录页面的"忘记密码"
2. 输入注册手机号或邮箱
3. 获取验证码或查收重置邮件
4. 输入验证码或点击邮件链接
5. 设置新密码
6. 确认新密码
7. 完成重置

**注意事项：**
- 新密码需符合安全要求
- 建议设置强密码
- 重置后请及时登录确认

### 登录时提示账户不存在？

**可能原因：**
- 手机号或邮箱输入错误
- 使用了不同的注册方式
- 账户可能被注销
- 记忆错误

**解决方案：**
1. 确认输入信息正确
2. 尝试其他注册方式登录
3. 检查是否使用了其他手机号或邮箱
4. 联系客服查询账户状态
5. 如确实不存在，重新注册

### 登录时提示密码错误？

**解决步骤：**
1. 确认密码输入正确
2. 检查大小写锁定状态
3. 确认输入法状态
4. 尝试复制粘贴密码
5. 使用忘记密码功能重置
6. 尝试验证码登录

**安全提示：**
- 连续输错可能导致账户临时锁定
- 锁定后请等待或联系客服
- 建议使用密码管理器

### 账户被锁定怎么办？

**锁定原因：**
- 连续多次输入错误密码
- 异常登录行为
- 安全策略触发
- 违规操作

**解锁方法：**
1. **自动解锁**：等待锁定时间结束（通常15-30分钟）
2. **验证码解锁**：通过手机验证码验证身份
3. **邮箱解锁**：通过邮箱验证解锁
4. **客服解锁**：联系客服人工处理

**预防措施：**
- 使用正确的登录信息
- 避免在公共设备上登录
- 定期更换密码
- 启用双因素认证

## 账户安全

### 如何设置强密码？

**密码要求：**
- 长度至少8位，建议12位以上
- 包含大写字母（A-Z）
- 包含小写字母（a-z）
- 包含数字（0-9）
- 包含特殊字符（!@#$%^&*等）

**密码建议：**
- 避免使用个人信息
- 不要使用常见密码
- 定期更换密码
- 不同平台使用不同密码
- 使用密码管理器

**示例：**
- 弱密码：123456、password、abc123
- 强密码：MyP@ssw0rd2025!、Zai#2025$Secure

### 如何启用双因素认证？

**启用步骤：**
1. 登录账户后进入"安全设置"
2. 找到"双因素认证"选项
3. 选择认证方式：
   - 短信验证码
   - 邮箱验证码
   - 认证器应用（推荐）
4. 按照提示完成设置
5. 保存备用恢复码

**认证器应用推荐：**
- Google Authenticator
- Microsoft Authenticator
- Authy
- 1Password

**重要提醒：**
- 妥善保管备用恢复码
- 定期检查认证设置
- 更换手机时及时更新

### 如何保护账户安全？

**安全建议：**

1. **密码安全**
   - 使用强密码
   - 定期更换密码
   - 不要共享密码
   - 使用密码管理器

2. **登录安全**
   - 启用双因素认证
   - 避免在公共设备登录
   - 及时注销登录
   - 关注登录异常提醒

3. **环境安全**
   - 使用安全的网络环境
   - 避免在公共WiFi下操作
   - 保持设备系统更新
   - 安装安全软件

4. **信息保护**
   - 不要泄露个人信息
   - 谨慎处理钓鱼邮件
   - 定期检查账户活动
   - 及时报告异常情况

## 账户管理

### 如何修改个人信息？

**修改步骤：**
1. 登录账户
2. 进入"个人中心"
3. 点击"个人信息"
4. 修改需要更新的信息
5. 验证身份（如需要）
6. 保存修改

**可修改信息：**
- 昵称
- 头像
- 个人简介
- 联系方式
- 公司信息

**注意事项：**
- 部分信息修改需要验证
- 重要信息修改可能有限制
- 修改后请确认信息正确

### 如何更换绑定手机号？

**更换步骤：**
1. 进入"安全设置"
2. 找到"手机号管理"
3. 点击"更换手机号"
4. 验证当前手机号
5. 输入新手机号
6. 验证新手机号
7. 确认更换

**验证方式：**
- 短信验证码
- 语音验证码
- 邮箱验证（备用）

**注意事项：**
- 确保新手机号可正常使用
- 更换后及时更新相关设置
- 保留更换记录

### 如何更换绑定邮箱？

**更换流程：**
1. 进入"安全设置"
2. 选择"邮箱管理"
3. 点击"更换邮箱"
4. 验证当前邮箱
5. 输入新邮箱地址
6. 查收验证邮件
7. 点击邮件中的确认链接
8. 完成更换

**邮箱要求：**
- 有效的邮箱地址
- 能够正常接收邮件
- 未被其他账户绑定

### 如何注销账户？

**注销前准备：**
- 备份重要数据
- 处理未完成的订单
- 确认账户余额
- 了解注销后果

**注销步骤：**
1. 进入"账户设置"
2. 找到"账户注销"
3. 阅读注销说明
4. 确认注销条件
5. 验证身份
6. 提交注销申请
7. 等待审核处理

**注销条件：**
- 无未完成的订单
- 无未结清的费用
- 无违规行为记录
- 满足冷静期要求

**注销后果：**
- 账户数据将被删除
- 无法恢复账户
- 失去所有权益
- 手机号/邮箱可重新注册

## 常见错误解决

### 网络连接问题

**错误现象：**
- 页面加载缓慢
- API调用超时
- 连接中断

**解决方案：**
1. 检查网络连接状态
2. 尝试刷新页面
3. 更换网络环境
4. 检查防火墙设置
5. 联系网络服务商

### 浏览器兼容性问题

**推荐浏览器：**
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

**解决方案：**
1. 更新浏览器到最新版本
2. 清除浏览器缓存
3. 禁用浏览器插件
4. 尝试无痕模式
5. 更换推荐浏览器

### API调用错误

**常见错误码：**
- 401：认证失败
- 403：权限不足
- 429：请求频率超限
- 500：服务器内部错误

**解决方案：**
1. 检查API密钥是否正确
2. 确认账户权限和余额
3. 控制请求频率
4. 检查请求参数格式
5. 查看详细错误信息
