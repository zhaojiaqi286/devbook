# 费用问题

本页面汇总了用户在使用Z.AI大模型开放平台时遇到的费用相关问题及解决方案。

## 计费方式

### 开放平台如何计费？

**计费单位：**
- 我们的计费方式是以 token 为单位（图像大模型按照模型产出的图片数量计费）
- 每个 token 代表一个自然语言文本的基本单位，如"字"或"词"

**计费范围：**
- 我们会根据您的模型输入和输出的总 token 数进行计费
- 向量大模型embedding-2仅按照输入token量计费
- 如果您开启了搜索服务，搜索结果作为输入也会被计费

**扣减方式：**
我们支持两种扣减方式：
1. **费用扣减**：根据 token 使用量×模型单价从您的现金余额账户中扣除
2. **资源包扣减**：根据 token 使用量从您的资源包账户扣除

**扣减顺序：**
在扣除时，我们会优先扣除资源包账户，然后再扣除现金余额账户。模型单价可参考产品定价。

## 资源包管理

### 资源包有扣费顺序吗？

**扣费规则：**
- 您调用模型时，优先扣除满足模型适用场景的资源包余额，再扣除现金账户余额
- 存在多个相同适用场景的资源包时，将优先扣除最快过期的资源包

### 欠费状态下可以使用资源包吗？

您好，欠费状态下可以使用有效期内的资源包。

### 资源包到期能否续期？

资源包过期后，将不支持延期、续费、重新激活，请您关注资源包过期时间，并及时使用。

## 体验中心

### 体验中心是否计费？

体验中心的计费规则和 API 调用一致，具体的模型调用单价可参考产品定价。

## 消费查询

### 如何查看我的消费明细？

**查看方式：**
1. **财务总览页面**：查看消耗情况，包括今日消费金额以及近6个月的消费统计
2. **费用账单页面**：查看详细的使用记录
3. **导出记录页面**：点击下载汇总账单

**可查看信息：**
- 实时消费金额
- 历史消费趋势
- 详细使用记录
- 按模型分类的消费

## 特殊计费

### 图片怎么计算tokens？

您好，调用GLM-4V系列图像识别模型时，单张图片大约消耗1047个tokens。

### 平台的apikey删除后，还会扣钱吗？

如果APIkey被删除，则该APIkey将无法再次成功调用接口，也不会产生扣费。

## 退款政策

### 如何退款？

Bigmodel开放平台暂时不支持任意形式的退款功能，请您充分预估实际需求并确定充值金额，由于平台接口对tokens连续消耗的特性，消耗账户通常不支持退款。

**特殊情况退款：**
如您因特殊情况必须退款，尚未消费的充值金额（不含体验金、优惠金额等）可回复"人工"，联系平台客服发起特殊申请。

**退款流程：**
1. 联系客服发起申请
2. 提供相关证明材料
3. 平台审核退款事宜
4. 审核通过后原路返还

**退款注意事项：**
- 一旦发起退款申请，充值优惠金额将于退款申请通过之时失效
- 您的充值优惠金额已经使用，您又对该笔充值发起全部或部分退款的，在退款时我们将全额从充值金额中扣除已经使用的充值优惠金额
- 您完成充值并已经消耗的或根据相关产品（或服务）协议应予扣除的，不支持退款
- 已开票部分的充值金额不支持退款

## 费用优化建议

### 如何优化使用成本？

**成本优化策略：**
1. **选择合适的模型**：根据需求选择性价比最高的模型
2. **使用资源包**：批量购买资源包享受优惠价格
3. **优化prompt**：精简输入内容，减少不必要的token消耗
4. **使用Batch API**：批量处理享受50%价格优惠
5. **监控使用量**：定期查看消费明细，及时调整使用策略

**节省技巧：**
- 避免重复调用
- 合理设置参数
- 充分利用缓存
- 选择合适的调用方式
