# 实名认证问题

本页面汇总了用户在进行Z.AI大模型开放平台实名认证时遇到的常见问题及解决方案。

## 认证类型

### 个人认证账号和企业认证账号有什么区别吗？

企业认证账号和个人认证账号认证方式不同，提交的认证材料也不同。

**企业认证账号优势：**
- 完成企业认证的开发者可享有智谱企业权益，详见：用户权益
- 完成企业认证的开发者，可以通过对公打款方式进行账号充值
- 我们强烈建议企业账号不要使用个人身份进行个人实名认证，以免企业人员变动或交接引发账号登录信息丢失或产生不必要的纠纷

### 没有实名认证，可以使用 API 吗？

目前调用 API 并不强制要求实名认证，但为确保您的账户安全，我们建议您进行实名认证。

## 认证变更

### 企业实名账号可以更改为个人账号吗？

企业实名账号不支持更改为个人账号，敬请谅解。

### 个人账号可以更改为企业账号吗？

个人实名账号可以变更为企业实名账号，您可以访问实名认证页面，点击"变更为企业认证"按钮，变更您的账号认证类型即可。

## 海外认证

### 海外企业是否可以进行企业认证？

平台是支持海外企业进行认证的，您需要在实名认证页面选择海外企业并上传相关的认证材料即可。

## 认证数量

### 每个企业/每个人可以认证多少个账号？

每个企业/每个人可以实名认证多个账号，目前没有认证数量限制。

## 审核时间

### 实名认证提交后，需要等待多久可以审核完成？

**不同认证方式的审核时间：**

- **个人人脸识别认证**：人脸识别通过后，即时完成
- **企业法人人脸识别认证**：7个工作日内
- **企业营业执照授权认证**：7个工作日内
- **企业对公打款认证**：最快30分钟

**注意事项：**
- 请确保提交的认证材料清晰、完整
- 认证信息需与实际情况一致
- 如有疑问，可联系客服咨询
