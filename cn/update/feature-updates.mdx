## 最新功能更新

### 2025年4月 - AI 搜索工具全家桶升级

**一站式 AI 搜索工具全家桶更新升级**

全系列 AI 搜索工具升级上线，覆盖三大核心服务：

#### 🔍 基础检索（Web Search API）
- 支持直接获取结构化搜索结果（标题/摘要/链接等）
- 提供多搜索引擎支持（智谱自研/bing/搜狗/夸克/Jina AI）
- 灵活切换搜索引擎
- 优化搜索结果准确度

#### 💬 问答增强（Web Search in Chat）
- 将 Web Search API 的搜索结果融入大模型
- 生成智能回答
- 标注网页结果来源
- 提升回答的时效性和准确性

#### 🤖 搜索智能体（Search Agent）
- 根据用户的 query，分析搜索意图
- 进行多个 query 拆解
- 将搜索结果融入大模型
- 提供全面、有深度的回答

**技术改进：**
- 新增支持 bing、搜狗、夸克、Jina.ai 等主流搜索引擎
- 优化了意图识别和结果搜索的准确度
- 上新限时特惠价格
