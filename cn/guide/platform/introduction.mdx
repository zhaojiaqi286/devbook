## 核心能力

### 1. 大语言模型服务

#### GLM-4系列模型

**GLM-4-Plus**
- 最强性能的旗舰模型
- 支持128K上下文长度
- 优秀的推理和创作能力
- 适用于复杂任务处理

**GLM-4-0520**
- 平衡性能和成本
- 快速响应能力
- 适用于大多数应用场景
- 高性价比选择

**GLM-4-Air**
- 轻量级模型
- 超快响应速度
- 适用于实时交互
- 成本优化方案

**GLM-4-Flash**
- 极速推理模型
- 毫秒级响应
- 适用于高并发场景
- 最优成本效益

#### 特色功能

```python
# 多轮对话
response = client.chat.completions.create(
    model="glm-4-plus",
    messages=[
        {"role": "user", "content": "什么是人工智能？"},
        {"role": "assistant", "content": "人工智能是..."},
        {"role": "user", "content": "它有哪些应用？"}
    ]
)

# 长文本处理
response = client.chat.completions.create(
    model="glm-4-plus",
    messages=[
        {"role": "user", "content": f"请总结这篇长文档：{long_document}"}
    ]
)

# 结构化输出
response = client.chat.completions.create(
    model="glm-4-plus",
    messages=[
        {"role": "user", "content": "分析这个商业案例"}
    ],
    response_format={"type": "json_object"}
)
```

### 2. 多模态AI能力

#### 视觉理解

**GLM-4V系列**
- 图像理解和分析
- 图文混合对话
- OCR文字识别
- 图表数据提取

```python
# 图像分析
response = client.chat.completions.create(
    model="glm-4v-plus",
    messages=[
        {
            "role": "user",
            "content": [
                {"type": "text", "text": "这张图片显示了什么？"},
                {"type": "image_url", "image_url": {"url": "data:image/jpeg;base64,..."}}
            ]
        }
    ]
)
```

#### 图像生成

**CogView系列**
- 文本到图像生成
- 多种艺术风格
- 高分辨率输出
- 批量生成支持

```python
# 图像生成
response = client.images.generations(
    model="cogview-3-plus",
    prompt="一只可爱的熊猫在竹林中玩耍，水彩画风格",
    size="1024x1024",
    quality="standard",
    n=1
)
```

#### 视频生成

**CogVideoX**
- 文本到视频生成
- 图像到视频转换
- 多种分辨率支持
- 自定义时长控制

```python
# 视频生成
response = client.videos.generations(
    model="cogvideox",
    prompt="海边日落，海浪轻拍沙滩，宁静美好",
    image_url="https://example.com/image.jpg",  # 可选的起始图像
    duration=5  # 视频时长（秒）
)
```

### 4. 智能体开发

#### 核心组件

**函数调用（Function Calling）**
```python
# 定义工具函数
tools = [
    {
        "type": "function",
        "function": {
            "name": "get_weather",
            "description": "获取指定城市的天气信息",
            "parameters": {
                "type": "object",
                "properties": {
                    "city": {
                        "type": "string",
                        "description": "城市名称"
                    }
                },
                "required": ["city"]
            }
        }
    }
]

# 使用工具
response = client.chat.completions.create(
    model="glm-4-plus",
    messages=[{"role": "user", "content": "北京今天天气怎么样？"}],
    tools=tools,
    tool_choice="auto"
)
```

**网络搜索**
```python
# 启用网络搜索
response = client.chat.completions.create(
    model="glm-4-plus",
    messages=[
        {"role": "user", "content": "最新的人工智能发展趋势是什么？"}
    ],
    tools=[{"type": "web_search"}]
)
```

**代码执行**
```python
# 代码解释器
response = client.chat.completions.create(
    model="glm-4-plus",
    messages=[
        {"role": "user", "content": "帮我分析这个数据集并生成可视化图表"}

    tools=[{"type": "code_interpreter"}]
)
```

**文档检索**

## 开始使用

### 1. 注册账号

1. 访问 [智谱AI开放平台](https://bigmodel.cn)
2. 点击"注册"创建账号
3. 完成邮箱验证
4. 实名认证（企业用户需要企业认证）

### 2. 获取API密钥

1. 登录控制台
2. 进入"API密钥"页面
3. 点击"创建新密钥"
4. 复制并安全保存密钥

### 3. 快速开始

```python
# 安装SDK
pip install zhipuai

# 第一个API调用
from zhipuai import ZhipuAI

client = ZhipuAI(api_key="your_api_key")

response = client.chat.completions.create(
    model="glm-4-flash",
    messages=[
        {"role": "user", "content": "你好，智谱AI！"}
    ]
)

print(response.choices[0].message.content)
```
