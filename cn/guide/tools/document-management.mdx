## 使用方式

<Tabs>
  <Tab title="文档上传">

### 文档上传

```python
from zhipuai import ZhipuAI

client = ZhipuAI(api_key="your_api_key")

# 上传文档
response = client.files.create(
    file=open("document.pdf", "rb"),
    purpose="retrieval"
)

file_id = response.id
print(f"文档上传成功，文件ID: {file_id}")
```

  </Tab>
  <Tab title="创建知识库">

### 创建知识库

```python
# 创建知识库
knowledge_base = client.knowledge.create(
    name="企业知识库",
    description="包含公司政策、流程和FAQ的知识库"
)

kb_id = knowledge_base.id
```

  </Tab>
  <Tab title="添加文档">

### 添加文档到知识库

```python
# 将文档添加到知识库
client.knowledge.documents.create(
    knowledge_base_id=kb_id,
    file_id=file_id
)
```

  </Tab>
  <Tab title="文档检索">

### 文档检索

```python
# 在知识库中搜索
results = client.knowledge.search(
    knowledge_base_id=kb_id,
    query="公司的休假政策是什么？",
    top_k=5
)

for result in results:
    print(f"相关度: {result.score}")
    print(f"内容: {result.content}")
    print(f"来源: {result.metadata.filename}")
    print("---")
```

  </Tab>
</Tabs>

## 最佳实践

<Accordion title="文档准备">
1. **格式规范**：确保文档格式清晰，结构合理
2. **内容质量**：提供准确、完整的信息
3. **文件命名**：使用有意义的文件名
4. **版本管理**：及时更新过时的文档
</Accordion>

<Accordion title="知识库组织">
1. **分类管理**：按主题或部门组织知识库
2. **权限控制**：设置适当的访问权限
3. **定期维护**：清理无用文档，更新内容
4. **标签标注**：为文档添加相关标签
</Accordion>

<Accordion title="检索优化">
1. **查询优化**：使用清晰、具体的查询语句
2. **结果筛选**：根据相关度和来源筛选结果
3. **上下文利用**：结合上下文信息提高检索准确性
4. **反馈改进**：根据使用反馈优化知识库
</Accordion>
