## 什么是 Z.ai Java SDK

Z.ai Java SDK 是一个功能强大、易于使用的Java开发工具包，专为与智谱AI的各种人工智能模型进行交互而设计。无论您是要构建企业级AI应用、智能客服系统，还是需要在现有Java应用中集成AI功能，我们的SDK都能为您提供完整的解决方案。

### 支持的功能

- **💬 对话聊天**：支持单轮和多轮对话，流式和非流式响应
- **🔧 函数调用**：让AI模型调用您的自定义函数
- **👁️ 视觉理解**：图像分析、OCR、视觉问答
- **🎨 图像生成**：根据文本描述生成高质量图像
- **🎬 视频生成**：文本到视频的创意内容生成
- **🔊 语音处理**：语音转文字、文字转语音
- **📊 文本嵌入**：文本向量化，支持语义搜索
- **🤖 智能助手**：构建专业的AI助手应用
- **📚 知识库管理**：文档上传、检索、问答
- **🛡️ 内容审核**：文本和图像内容安全检测

## 技术规格

### 环境要求

- **Java版本**：Java 1.8 或更高版本
- **构建工具**：Maven 3.6+ 或 Gradle 6.0+
- **网络要求**：支持HTTPS连接
- **API密钥**：需要有效的智谱AI API密钥

### 依赖管理

SDK采用模块化设计，您可以根据需要选择性引入功能模块：

- **核心模块**：基础API调用功能
- **异步模块**：异步和并发处理支持
- **工具模块**：实用工具和辅助功能

#### 基础对话

```java
import com.zhipuai.ZaiClient;
import com.zhipuai.model.ChatCompletionRequest;
import com.zhipuai.model.ChatCompletionResponse;
import com.zhipuai.model.ChatMessage;
import java.util.Arrays;

public class BasicChat {
    public static void main(String[] args) {
        // 初始化客户端
        ZaiClient client = ZaiClient.builder()
            .apiKey("your-api-key")
            .build();
        
        // 创建聊天完成请求
        ChatCompletionRequest request = ChatCompletionRequest.builder()
            .model("glm-4")
            .messages(Arrays.asList(
                ChatMessage.userMessage("你好，请介绍一下自己")
            ))
            .build();
        
        // 发送请求
        ChatCompletionResponse response = client.chatCompletions().create(request);
        
        // 获取回复
        String reply = response.getChoices().get(0).getMessage().getContent();
        System.out.println("AI回复: " + reply);
    }
}
```

#### 流式对话

```java
import com.zhipuai.ZaiClient;
import com.zhipuai.model.ChatCompletionRequest;
import com.zhipuai.model.ChatCompletionChunk;
import com.zhipuai.model.ChatMessage;
import java.util.Arrays;

public class StreamingChat {
    public static void main(String[] args) {
        ZaiClient client = ZaiClient.builder()
            .apiKey("your-api-key")
            .build();
        
        // 创建流式聊天请求
        ChatCompletionRequest request = ChatCompletionRequest.builder()
            .model("glm-4")
            .messages(Arrays.asList(
                ChatMessage.userMessage("写一首关于春天的诗")
            ))
            .stream(true)
            .build();
        
        // 处理流式响应
        client.chatCompletions().createStream(request)
            .forEach(chunk -> {
                if (chunk.getChoices() != null && !chunk.getChoices().isEmpty()) {
                    String content = chunk.getChoices().get(0).getDelta().getContent();
                    if (content != null) {
                        System.out.print(content);
                    }
                }
            });
        
        System.out.println(); // 换行
    }
}
```

### 完整示例

```java
import com.zhipuai.ZaiClient;
import com.zhipuai.model.*;
import java.util.*;
import java.util.stream.Collectors;

public class ChatBot {
    private final ZaiClient client;
    private final List<ChatMessage> conversation;
    
    public ChatBot(String apiKey) {
        this.client = ZaiClient.builder()
            .apiKey(apiKey)
            .build();
        this.conversation = new ArrayList<>();
        // 添加系统消息
        this.conversation.add(ChatMessage.systemMessage("你是一个友好的AI助手"));
    }
    
    public String chat(String userInput) {
        try {
            // 添加用户消息
            conversation.add(ChatMessage.userMessage(userInput));
            
            // 创建请求
            ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("glm-4")
                .messages(conversation)
                .temperature(0.7)
                .maxTokens(1000)
                .build();
            
            // 发送请求
            ChatCompletionResponse response = client.chatCompletions().create(request);
            
            // 获取AI回复
            String aiResponse = response.getChoices().get(0).getMessage().getContent();
            
            // 添加AI回复到对话历史
            conversation.add(ChatMessage.assistantMessage(aiResponse));
            
            return aiResponse;
            
        } catch (Exception e) {
            return "发生错误: " + e.getMessage();
        }
    }
    
    public static void main(String[] args) {
        ChatBot bot = new ChatBot(System.getenv("ZAI_API_KEY"));
        Scanner scanner = new Scanner(System.in);
        
        System.out.println("欢迎使用 Z.ai 聊天机器人！输入 'quit' 退出。");
        
        while (true) {
            System.out.print("您: ");
            String input = scanner.nextLine();
            
            if ("quit".equalsIgnoreCase(input)) {
                break;
            }
            
            String response = bot.chat(input);
            System.out.println("AI: " + response);
        }
        
        System.out.println("再见！");
        scanner.close();
    }
}
```

## 高级功能

### 函数调用 (Function Calling)

函数调用允许 AI 模型调用您定义的函数来获取实时信息或执行特定操作。

#### 定义和使用函数

```java
import com.zhipuai.ZaiClient;
import com.zhipuai.model.*;
import com.fasterxml.jackson.databind.ObjectMapper;
import java.util.*;

public class FunctionCallingExample {
    private static final ObjectMapper objectMapper = new ObjectMapper();
    
    // 模拟天气API
    public static Map<String, Object> getWeather(String location, String date) {
        Map<String, Object> weather = new HashMap<>();
        weather.put("location", location);
        weather.put("date", date != null ? date : "今天");
        weather.put("weather", "晴天");
        weather.put("temperature", "25°C");
        weather.put("humidity", "60%");
        return weather;
    }
    
    // 模拟股票API
    public static Map<String, Object> getStockPrice(String symbol) {
        Map<String, Object> stock = new HashMap<>();
        stock.put("symbol", symbol);
        stock.put("price", 150.25);
        stock.put("change", "+2.5%");
        return stock;
    }
    
    public static void main(String[] args) {
        ZaiClient client = ZaiClient.builder()
            .apiKey(System.getenv("ZAI_API_KEY"))
            .build();
        
        // 定义函数工具
        List<Tool> tools = Arrays.asList(
            Tool.function(
                Function.builder()
                    .name("get_weather")
                    .description("获取指定地点的天气信息")
                    .parameters(Parameters.builder()
                        .type("object")
                        .properties(Map.of(
                            "location", Property.builder()
                                .type("string")
                                .description("地点名称")
                                .build(),
                            "date", Property.builder()
                                .type("string")
                                .description("日期，格式为YYYY-MM-DD")
                                .build()
                        ))
                        .required(Arrays.asList("location"))
                        .build())
                    .build()
            ),
            Tool.function(
                Function.builder()
                    .name("get_stock_price")
                    .description("获取股票当前价格")
                    .parameters(Parameters.builder()
                        .type("object")
                        .properties(Map.of(
                            "symbol", Property.builder()
                                .type("string")
                                .description("股票代码")
                                .build()
                        ))
                        .required(Arrays.asList("symbol"))
                        .build())
                    .build()
            )
        );
        
        // 创建请求
        ChatCompletionRequest request = ChatCompletionRequest.builder()
            .model("glm-4")
            .messages(Arrays.asList(
                ChatMessage.userMessage("北京今天天气怎么样？")
            ))
            .tools(tools)
            .toolChoice("auto")
            .build();
        
        // 发送请求
        ChatCompletionResponse response = client.chatCompletions().create(request);
        
        // 处理函数调用
        ChatMessage assistantMessage = response.getChoices().get(0).getMessage();
        if (assistantMessage.getToolCalls() != null && !assistantMessage.getToolCalls().isEmpty()) {
            for (ToolCall toolCall : assistantMessage.getToolCalls()) {
                String functionName = toolCall.getFunction().getName();
                String arguments = toolCall.getFunction().getArguments();
                
                try {
                    Map<String, Object> args = objectMapper.readValue(arguments, Map.class);
                    Map<String, Object> result;
                    
                    if ("get_weather".equals(functionName)) {
                        result = getWeather(
                            (String) args.get("location"),
                            (String) args.get("date")
                        );
                        System.out.println("天气信息: " + result);
                    } else if ("get_stock_price".equals(functionName)) {
                        result = getStockPrice((String) args.get("symbol"));
                        System.out.println("股票信息: " + result);
                    }
                } catch (Exception e) {
                    System.err.println("函数调用错误: " + e.getMessage());
                }
            }
        } else {
            System.out.println(assistantMessage.getContent());
        }
    }
}
```

```java
import com.zhipuai.ZaiClient;
import com.zhipuai.model.*;
import java.util.*;
import java.nio.file.Files;
import java.nio.file.Paths;
import java.util.Base64;

public class ImageUnderstanding {
    public static void main(String[] args) throws Exception {
        ZaiClient client = ZaiClient.builder()
            .apiKey(System.getenv("ZAI_API_KEY"))
            .build();
        
        // 方式1：使用图像URL
        ChatCompletionRequest request1 = ChatCompletionRequest.builder()
            .model("glm-4v")
            .messages(Arrays.asList(
                ChatMessage.builder()
                    .role("user")
                    .content(Arrays.asList(
                        Content.text("这张图片里有什么？请详细描述。"),
                        Content.imageUrl("https://example.com/image.jpg")
                    ))
                    .build()
            ))
            .build();
        
        ChatCompletionResponse response1 = client.chatCompletions().create(request1);
        System.out.println(response1.getChoices().get(0).getMessage().getContent());
        
        // 方式2：使用base64编码的图像
        byte[] imageBytes = Files.readAllBytes(Paths.get("path/to/your/image.jpg"));
        String base64Image = Base64.getEncoder().encodeToString(imageBytes);
        
        ChatCompletionRequest request2 = ChatCompletionRequest.builder()
            .model("glm-4v")
            .messages(Arrays.asList(
                ChatMessage.builder()
                    .role("user")
                    .content(Arrays.asList(
                        Content.text("分析这张图片中的内容"),
                        Content.imageUrl("data:image/jpeg;base64," + base64Image)
                    ))
                    .build()
            ))
            .build();
        
        ChatCompletionResponse response2 = client.chatCompletions().create(request2);
        System.out.println(response2.getChoices().get(0).getMessage().getContent());
    }
}
```

### 异步编程

```java
import com.zhipuai.ZaiClient;
import com.zhipuai.model.*;
import java.util.concurrent.CompletableFuture;
import java.util.concurrent.ExecutionException;
import java.util.*;

public class AsyncExample {
    public static void main(String[] args) throws ExecutionException, InterruptedException {
        ZaiClient client = ZaiClient.builder()
            .apiKey(System.getenv("ZAI_API_KEY"))
            .build();
        
        // 异步聊天请求
        ChatCompletionRequest request = ChatCompletionRequest.builder()
            .model("glm-4")
            .messages(Arrays.asList(
                ChatMessage.userMessage("你好")
            ))
            .build();
        
        CompletableFuture<ChatCompletionResponse> future = 
            client.chatCompletions().createAsync(request);
        
        // 异步处理结果
        future.thenAccept(response -> {
            String reply = response.getChoices().get(0).getMessage().getContent();
            System.out.println("异步回复: " + reply);
        }).exceptionally(throwable -> {
            System.err.println("异步请求失败: " + throwable.getMessage());
            return null;
        });
        
        // 等待完成
        future.get();
        
        // 并发处理多个请求
        List<String> questions = Arrays.asList(
            "什么是人工智能？",
            "Java有什么优势？",
            "如何学习机器学习？"
        );
        
        List<CompletableFuture<ChatCompletionResponse>> futures = new ArrayList<>();
        
        for (String question : questions) {
            ChatCompletionRequest req = ChatCompletionRequest.builder()
                .model("glm-4")
                .messages(Arrays.asList(ChatMessage.userMessage(question)))
                .build();
            
            futures.add(client.chatCompletions().createAsync(req));
        }
        
        // 等待所有请求完成
        CompletableFuture<Void> allFutures = CompletableFuture.allOf(
            futures.toArray(new CompletableFuture[0])
        );
        
        allFutures.thenRun(() -> {
            for (int i = 0; i < futures.size(); i++) {
                try {
                    ChatCompletionResponse response = futures.get(i).get();
                    String answer = response.getChoices().get(0).getMessage().getContent();
                    System.out.println("问题" + (i + 1) + "的回答: " + answer);
                } catch (Exception e) {
                    System.err.println("获取回答失败: " + e.getMessage());
                }
            }
        }).get();
    }
}
```
