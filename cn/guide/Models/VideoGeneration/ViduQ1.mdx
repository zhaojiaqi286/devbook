## <Icon icon="arrow-down-big-small" iconType="solid" color="#3890ff" size={28} />  能力说明

<CardGroup cols={1}>
  <Card title="图生视频" icon="image">
    输入首帧图片以及对应的文本描述，生成视频。
  </Card>
  <Card title="首尾帧" icon="frame">
    支持输入两张图，上传的第一张图片视作首帧图，第二张图片视作尾帧图，模型将以此参数中传入的图片来生成视频。
  </Card>
  <Card title="文生视频" icon="text">
    输入一段文本，生成视频；目前支持通用风格以及专门优化的动漫风格。
  </Card>
</CardGroup>

<Tip>
  模型生成的视频 url 链接有效期为1天，如有需要请尽快转存。
</Tip>

## <Icon icon="stars" iconType="solid" color="#3890ff" size={28} />   推荐场景

<AccordionGroup>
  <Accordion title="影视生成" icon="camera-movie">
    - 输入剧本片段、概念设计图等素材，即可生成影视宣发片、特效镜头及影片辅助素材
    - 院线级的清晰度和画质质感，完整的画面细节
    - 专业级视频转场，画面间自然过渡
  </Accordion>
  <Accordion title="动漫制作" icon="eyes">
    - 输入角色设定稿、分镜脚本，快速输出二次元动态动画、风格化动漫短片
    - 支持国漫、日漫等风格
    - 对经典 IP 进行剧情延展与创意再生
  </Accordion>
  <Accordion title="短剧生产" icon="video">
    - 输入小说章节、短剧剧本，自动生成短视频、微短剧内容
    - 覆盖言情、悬疑、古装等多元题材
    - 适配多平台分发需求
  </Accordion>
  <Accordion title="广告营销" icon="computer-classic">
    - 根据产品图片与特点描述，快速生成极具吸引力的品牌广告、电商带货视频、互动类广告（如虚拟试穿）
    - 支持多平台尺寸与创意形式适配
  </Accordion>
  <Accordion title="文旅宣传" icon="map-location-dot">
    - 输入景点照片、文旅宣传文案，一键生成沉浸式文旅宣传片、旅行 MV
    - 结合文化 IP（如历史文物、城市符号），打造互动式数字体验内
  </Accordion>
</AccordionGroup>

## <Icon icon="rectangle-code" iconType="solid" color="#3890ff" size={28} />   调用示例

<Tabs>
  <Tab title="图生视频">
    ```SQL
    curl --location --request POST 'https://open.bigmodel.cn/api/paas/v4/videos/generations' \
    --header 'Authorization: {you apikey}' \
    --header 'Content-Type: application/json' \
    --data-raw '{
    "model":"viduq1-image",
    "image_url":"https://example.com/path/to/your/image.jpg",
    "prompt":"比得兔开小汽车，游走在马路上，脸上的表情充满开心喜悦。",
    "duration":5,
    "size":"1920x1080",
    "movement_amplitude":"auto"
    }'
    ```
  </Tab>
  <Tab title="首尾帧">
    ```SQL
    curl --location --request POST 'https://open.bigmodel.cn/api/paas/v4/videos/generations' \
    --header 'Authorization: {you apikey}' \
    --header 'Content-Type: application/json' \
    --data-raw '{
    "model":"viduq1-start-end",
    "image_url":["https://example.com/path/to/your/image.jpg","https://example.com/path/to/your/image1.jpg"],
    "prompt":"比得兔开小汽车，游走在马路上，脸上的表情充满开心喜悦。",
    "duration":5,
    "size":"1920x1080",
    "movement_amplitude":"auto"
    }'
    ```
  </Tab>
  <Tab title="文生视频">
    ```SQL
    curl --location --request POST 'https://open.bigmodel.cn/api/paas/v4/videos/generations' \
    --header 'Authorization: {you apikey}' \
    --header 'Content-Type: application/json' \
    --data-raw '{
    "model": "viduq1-text",
    "style": "anime",
    "prompt": "比得兔开小汽车，游走在马路上，脸上的表情充满开心喜悦。",
    "duration": 5,
    "aspect_ratio": "16:9",
    "size": "1920x1080",
    "movement_amplitude": "auto"
    }'
    ```
