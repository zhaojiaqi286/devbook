## <Icon icon="rectangle-code" iconType="solid" color="#3890ff" size={28} />   调用示例

<Tabs>
  <Tab title="文生视频">
    ```Python
    from zhipuai import ZhipuAI
    client = ZhipuAI(api_key="") # 请填写您自己的APIKey
    
    response = client.videos.generations(
    model="cogvideox-3",
    prompt="比得兔开小汽车，游走在马路上，脸上的表情充满开心喜悦。",
    quality="quality",  # 输出模式，"quality"为质量优先，"speed"为速度优先
    with_audio=True,
    size="1920x1080",  # 视频分辨率，支持最高4K（如: "3840x2160"）
    fps=30,  # 帧率，可选为30或60
    )
    print(response)
    ```
  </Tab>
  <Tab title="图生视频">
    ```Python
    from zhipuai import ZhipuAI
    
    # 初始化客户端，请填写您自己的APIKey
    client = ZhipuAI(api_key="YOUR_API_KEY")
    
    # 定义图片的URL地址
    image_url = "https://example.com/path/to/your/image.jpg"  # 替换为您的图片URL地址
    
    # 调用视频生成接口
    response = client.videos.generations(
    model="cogvideox-3",  # 使用的视频生成模型
    image_url=image_url,  # 提供的图片URL地址或者 Base64 编码
    prompt="让画面动起来",
    quality="quality",  # 输出模式，"quality"为质量优先，"speed"为速度优先
    with_audio=True,
    size="1920x1080",  # 视频分辨率，支持最高4K（如: "3840x2160"）
    fps=30,  # 帧率，可选为30或60
    )
    
    # 打印返回结果
    print(response)
    ```
  </Tab>
  <Tab title="首尾帧生视频">
    ```Python
    from zhipuai import ZhipuAI
    
    # 初始化客户端，请填写您自己的APIKey
    client = ZhipuAI(api_key="YOUR_API_KEY")
    
    # 定义首帧和尾帧的URL地址
    first_frame_url = "https://example.com/path/to/first_frame.jpg"  # 替换为首帧图片URL
    last_frame_url = "https://example.com/path/to/last_frame.jpg"    # 替换为尾帧图片URL
    
    # 调用视频生成接口（假设支持 image_urls）
    response = client.videos.generations(
    model="cogvideox-3",  # 使用的视频生成模型
    image_urls=[first_frame_url, last_frame_url],  # 传入首帧和尾帧的URL列表
    prompt="让画面动起来",
    quality="quality",  # 输出模式，"quality"为质量优先，"speed"为速度优先
    with_audio=True,
    size="1920x1080",  # 视频分辨率，支持最高4K（如: "3840x2160"）
    fps=30,  # 帧率，可选为30或60
    )
    
    # 打印返回结果
    print(response)
    ```
