# Registration and Login Issues

This page summarizes common questions and solutions encountered by users when registering and logging into the Z.AI large model open platform. If your question is not listed here, please contact our customer service team.

## Account Registration

### How to register a Z.AI platform account?

**Registration Methods:**
1. **Mobile Phone Registration**
   - Visit https://bigmodel.cn
   - Click the "Register" button
   - Enter your mobile phone number
   - Get and enter the verification code
   - Set a login password
   - Agree to the user agreement and privacy policy
   - Complete registration

2. **WeChat QR Code Registration**
   - Visit the official website homepage
   - Click "WeChat QR Code Login"
   - Scan the QR code with WeChat
   - Confirm authorization in WeChat
   - Complete registration by providing necessary information

3. **Email Registration**
   - Select the email registration option
   - Enter a valid email address
   - Check for verification email
   - Click the verification link in the email
   - Set password to complete registration

### What to do when registration shows "mobile number already exists"?

**Possible Reasons:**
- The mobile number has already been registered
- Previously registered with this number but forgotten

**Solutions:**
1. Try to log in directly
2. If password is forgotten, use "Forgot Password" function to reset
3. Contact customer service to confirm account status
4. If a new account is really needed, use a different mobile number

### What to do if SMS verification code is not received?

**Common Causes and Solutions:**

1. **Network Delay**
   - Wait 1-2 minutes and check again
   - Ensure good mobile signal

2. **SMS Blocked**
   - Check mobile spam folder
   - Add numbers starting with 106 to whitelist
   - Turn off SMS blocking function

3. **Incorrect Mobile Number**
   - Confirm mobile number is entered correctly
   - Pay attention to area code and format

4. **Carrier Issues**
   - Contact mobile carrier for confirmation
   - Try using a different mobile number

5. **Frequent Requests**
   - Avoid clicking get verification code frequently
   - Wait 60 seconds before requesting again

**Alternative Solutions:**
- Use email registration
- Use WeChat QR code registration
- Contact customer service for manual assistance

## Account Login

### What login methods are supported?

**Login Methods:**
1. **Mobile Phone Login**
   - Enter registered mobile number
   - Enter password or verification code
   - Click login

2. **Email Login**
   - Enter registered email
   - Enter password
   - Click login

3. **WeChat QR Code Login**
   - Click WeChat login
   - Scan QR code
   - Confirm authorization in WeChat

4. **Third-party Login**
   - Support GitHub login
   - Support Google login
   - More methods continuously added

### What to do if password is forgotten?

**Password Reset Steps:**
1. Click "Forgot Password" on login page
2. Enter registered mobile number or email
3. Get verification code or check reset email
4. Enter verification code or click email link
5. Set new password
6. Confirm new password
7. Complete reset

**Notes:**
- New password must meet security requirements
- Recommend setting a strong password
- Please log in promptly after reset to confirm

## Account Security

### How to set a strong password?

**Password Requirements:**
- At least 8 characters long
- Include uppercase and lowercase letters
- Include numbers and special characters
- Avoid using personal information
- Avoid common passwords

**Security Recommendations:**
- Use unique passwords for different accounts
- Enable two-factor authentication
- Regularly update passwords
- Use password managers
- Avoid logging in on public devices
