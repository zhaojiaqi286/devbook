## <Icon icon="rectangle-code" iconType="solid" color="#3890ff" size={28} />   Calling Example

```Python
from zhipuai import ZhipuAI
client = ZhipuAI(api_key="") # Fill in your own APIKey
response = client.chat.completions.create(
    model="glm-4.1v-thinking-flashx",  # Fill in the model name to be called
    messages=[
       {
        "role": "user",
        "content": [
          {
            "type": "text",
            "text": "Please help me solve this problem, providing a detailed process and answer"
          },
          {
            "type": "image_url",
            "image_url": {
                "url" : "Enter the URL address of the image"
            }
          }
        ]
      }
    ]
)
print(response.choices[0].message)
```
